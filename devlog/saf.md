Just a fun side project for convenience and leisure. This is a log of scripting efforts to make the [Serendipity Arts Festival's](https://serendipityartsfestival.com) event schedule accessible and consumable.

2 weeks to go. Can spare a maximum upto an hour a day.

- Setup an importable python module.
- Download the programmes page.
- Extract event cards from the downloaded html mpage.
- Extract event details from the event card.
- Print all the extracted event details in a table.
